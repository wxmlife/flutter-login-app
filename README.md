# MyNight 🔐
一个基于 Flutter 3 + Material 3 设计的精美登录页面。

## ✨ 功能特性

- 🌐 **Material 3 设计** — 圆角输入框、渐变背景、现代化 UI
- ✅ **表单验证** — 用户名必填、密码长度校验
- 👁️ **密码可见切换** — 点击图标切换密码显示/隐藏
- 🔄 **登录加载状态** — 防止重复提交
- 📱 **响应式布局** — 适配不同屏幕尺寸
- 🔗 **第三方登录** — Google、Facebook、Apple 社交登录入口
- 🌙 **渐变装饰背景** — 视觉层次丰富
